You are an ML researcher tasked with selectively unlearning specific visual concepts from a fine-tuned vision model using mechanistic interpretability techniques.

## Background

**Machine unlearning** removes specific knowledge from a trained model while preserving all other capabilities. Naive approaches (gradient ascent on forget data, full retraining) either fail to truly erase knowledge or destroy the model. Recent work shows that **mechanistic approaches** — using interpretability to localize which components encode target knowledge, then surgically editing those components — produce more precise and robust unlearning.

Key techniques from the literature:
- **Representation Misdirection (RMU)**: Steer intermediate-layer activations toward random vectors for forget-set data while anchoring retain-set data
- **Task vector subtraction**: Compute the weight difference between a model that knows concept X and one that doesn't, then subtract it
- **Circuit-level ablation**: Use probing or activation analysis to find which attention heads/MLP layers encode the target concept, then ablate or edit those specific components
- **Activation steering**: Find a "concept direction" in activation space and project it out at inference time

The challenge: the model must genuinely forget — not just suppress surface outputs while retaining internal representations. And it must not suffer collateral damage to unrelated concepts.

  ```python
  import torch, open_clip, torch.nn as nn

  checkpoint = torch.load("./model/concept_classifier.pt")
  clip_model, _, preprocess = open_clip.create_model_and_transforms("ViT-B-16", pretrained="openai")

  class CLIPConceptClassifier(nn.Module):
      def __init__(self, visual, num_concepts):
          super().__init__()
          self.visual = visual
          self.head = nn.Linear(512, num_concepts)  # CLIP ViT-B/16 outputs 512-dim
      def forward(self, x):
          features = self.visual(x)
          if features.dim() == 3: features = features[:, 0, :]
          return self.head(features)

  model = CLIPConceptClassifier(clip_model.visual, checkpoint["num_concepts"])
  model.load_state_dict(checkpoint["state_dict"])
  ```

## Your Task

1. **Understand** the model: Evaluate its current accuracy on forget and retain concepts using the validation set. Establish baselines.

2. **Localize** which model components encode the forget concepts. You decide the approach:
   - Probe individual layers/heads to find where forget concepts are most represented
   - Use activation patching to causally verify which components matter
   - Analyze attention patterns or feature directions
   - Train sparse autoencoders and identify concept-specific features
   - Any other interpretability technique you choose

3. **Unlearn** the forget concepts by editing the model. You decide the method:
   - Ablate or modify specific attention heads/MLP layers
   - Project out concept-specific directions from weight matrices
   - Apply representation misdirection at identified layers
   - Subtract task vectors computed from forget-concept data
   - Fine-tune specific components with a targeted objective
   - Any combination of techniques

4. **Verify** your unlearning:
   - Forget concepts: model accuracy should drop significantly (ideally to chance ~50%)
   - Retain concepts: model accuracy should stay close to the original
   - The model should not be lobotomized — general feature extraction should still work
